# AutoPrime - Site de Venda de Automóveis

## 1. Contexto do Projeto
O AutoPrime é uma landing page responsiva para venda de automóveis desenvolvida com HTML e CSS. O projeto foi criado com foco em boas práticas de desenvolvimento web, acessibilidade, organização semântica e experiência do usuário em diferentes dispositivos.

## 2. Público-Alvo
O público-alvo são pessoas entre 25 e 50 anos interessadas na compra de veículos pela internet. O principal contexto de uso é a navegação rápida para visualizar modelos e entrar em contato com a empresa, principalmente por dispositivos móveis.

## 3. Dor Principal
Atualmente, muitos usuários encontram dificuldades para visualizar informações claras sobre veículos em sites desorganizados, pouco responsivos e difíceis de navegar em smartphones.

## 4. Critério de Sucesso
O usuário consegue visualizar os veículos disponíveis, navegar entre as seções do site e entrar em contato com a empresa de forma rápida e acessível em qualquer dispositivo.
